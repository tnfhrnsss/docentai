DocentAI - Privacy Policy (개인정보 처리방침)

Last Updated: January 20, 2026

요약: DocentAI는 넷플릭스 자막 설명을 제공하기 위해 최소한의 데이터만 수집하며, 개인 식별 정보는 수집하지 않습니다.

1. 수집하는 정보 (Information We Collect)

DocentAI Chrome Extension은 자막 설명 서비스를 제공하기 위해 다음 정보를 수집합니다:

  • 영상 메타데이터: 영상 제목, 에피소드 번호
  • 자막 텍스트: 사용자가 설명을 요청한 자막 및 전후 맥락 자막
  • 스크린샷 이미지: 사용자가 직접 화면 캡처 또는 파일 선택 기능을 통해 첨부한 이미지 (선택사항)

수집하지 않는 정보:

  • 개인 식별 정보 (이름, 이메일, 전화번호 등)
  • Netflix 계정 정보 (아이디, 비밀번호 등)
  • 결제 정보
  • 브라우징 히스토리
  • 쿠키 또는 추적 데이터

2. 정보 사용 목적 (How We Use Information)

수집된 정보는 다음 목적으로만 사용됩니다:

  • AI 기반 자막 설명 생성
  • 응답 캐싱을 통한 서비스 속도 개선
  • 서비스 품질 및 정확도 개선

3. 데이터 저장 및 보관 (Data Storage)

  • 로컬 스토리지: 사용자 설정 (언어, 플로팅 버튼 표시 여부 등)은 브라우저 로컬에만 저장
  • 이미지: 업로드된 이미지는 분석 후 자동 삭제 또는 최대 7일 보관

4. 제3자 공유 (Third-Party Sharing)

DocentAI는 수집된 데이터를 제3자와 공유하거나 판매하지 않습니다.

예외:

  • 법적 요구가 있는 경우 (법원 명령, 수사기관 요청 등)
  • 서비스 제공을 위해 필요한 AI API 제공자 (OpenAI, Anthropic 등)

5. 데이터 보안 (Data Security)

모든 데이터 전송은 HTTPS 암호화를 통해 보호되며, 서버는 업계 표준 보안 조치를 따릅니다.

6. 사용자 권리 (Your Rights)

사용자는 다음 권리를 가집니다:

  • 삭제 권한: Chrome Extension을 제거하면 로컬 데이터가 즉시 삭제됩니다
  • 서비스 중단: 언제든지 Extension 사용을 중단할 수 있습니다
  • 설정 변경: Extension 설정에서 기능을 활성화/비활성화할 수 있습니다

7. 아동 개인정보 보호 (Children's Privacy)

DocentAI는 만 13세 미만 아동을 대상으로 하지 않으며, 고의로 아동의 개인정보를 수집하지 않습니다.

8. 정책 변경 (Changes to Privacy Policy)

본 개인정보 처리방침은 서비스 개선을 위해 업데이트될 수 있습니다. 중요한 변경 사항이 있을 경우 Extension을 통해 공지합니다.

9. 문의 (Contact Us)

개인정보 처리방침에 대한 질문이나 우려사항이 있으시면 다음으로 연락주세요:

How We Use Information

Data is used solely for:

  • Generating AI-powered subtitle explanations
  • Improving service quality and speed

We do not sell or share your data with third parties, except for AI API providers necessary for service delivery.
